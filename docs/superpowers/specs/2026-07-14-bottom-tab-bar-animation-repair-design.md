# SwiftUI 底部导航栏动画修复设计

日期：2026-07-14

## 目标

仅重构 `FloatingTabBar` 的动画、图层和收起/展开状态切换，使完整玻璃胶囊自然退去，同时让同一个浅粉半透明活动玻璃控件移动并收拢为左下角 54pt 圆形按钮，再平滑反向展开。页面内容、业务数据、导航目标、图标顺序和整体布局风格保持不变。

## 已确认根因

1. `TabBarAnimationModel.itemState` 将所有未选中图标的位置插值到左侧同一个中心点，造成收起过程中的挤压、穿越、重叠和重影。
2. `navigationItems` 的宽度随胶囊一起缩小并使用 `.clipped()`，活动按钮及浅粉选中背景因此被横向裁切，产生粉色尾巴和矩形残片。
3. 玻璃背景、阴影、活动按钮和未选中图标使用不同响应时间与逐项延迟，动画中段拖沓、末段吸附。
4. 早期实现额外引入 `colorProgress`，把浅粉玻璃和红色图标改成实心红底白图标，破坏了同一活动控件的视觉连续性。
5. `CollapsingScrollView` 与 `FloatingTabBar` 同时施加不同弹簧，且滚动状态没有动画锁，快速反向滚动可能重复触发状态切换。

## 选定方案

### 单一持续活动按钮

活动按钮使用一个始终存在的视觉层，不创建展开版和收起版两个按钮，也不通过透明度交叉淡入淡出。它位于玻璃背景和普通图标层之上，只使用连续进度计算中心位置、宽高和圆角。

展开状态：

- 活动按钮位于当前标签槽位。
- 背景为现有浅粉色选中样式。
- 图标为品牌红色。

收起状态：

- 活动按钮中心移动到左侧 54pt 圆形区域中心。
- 背景尺寸为 54 × 54pt，继续使用与展开状态完全相同的浅粉半透明 Liquid Glass 样式。
- 图标继续使用同一个品牌红色并保持中心对齐。
- 仅保留轻微高光和轻量阴影。

活动背景只有一个几何来源，由统一的 `ActiveTabGlassStyle` 提供。色彩、材质透明度、饱和度、高光和图标颜色在整个进度中保持不变。

### 玻璃背景层

外围玻璃胶囊只负责自身尺寸和材质，从展开宽度连续缩小到零并退去。仅外围玻璃背景使用 Capsule 轮廓，不再裁切整个导航图标层。活动按钮使用自身的连续圆角形状并以较高 `zIndex` 放在玻璃背景之上。

### 未选中图标

未选中图标始终保留在展开状态的固定槽位，不再向左移动。收起时统一执行：

- 透明度从 1 降到 0。
- 缩放从 1 降到 0.92。
- 在不可见阶段关闭点击和辅助功能暴露。

展开时先启动胶囊几何动画。当胶囊已展开约 50% 后，再整体淡入普通图标，不使用明显的逐项弹出或不同方向位移。

### 动画时间线

所有几何属性共用：

```swift
.spring(
    response: 0.36,
    dampingFraction: 0.88,
    blendDuration: 0.08
)
```

收起目标时长约 0.34 秒，展开约 0.38 秒。活动背景与图标颜色不参与动画。普通图标仅使用短时透明度/缩放过渡，不参与几何移动。

Reduce Motion 开启时使用短暂 ease-out，移除弹簧和明显缩放，但保持状态、颜色和最终几何一致。

### 滚动状态控制

`TabBarScrollBehavior` 使用滞回阈值：

- 向下累计超过 20pt 后收起。
- 向上累计超过 16pt 后展开。
- 接近列表顶部时仍强制展开。

`CollapsingScrollView` 在触发切换后锁定约 0.4 秒。锁定期间仍更新基准滚动位置并清空累计距离，但不发起相反转换，避免解锁后因累计大位移立即反跳。外层不再给 `isCollapsed` 额外施加竞争弹簧，动画由 `FloatingTabBar` 唯一负责。

## 内容避让

现有 `TabContent.safeAreaPadding(.bottom, 76)` 已覆盖 54pt 按钮、4pt 底部间距和至少 18pt 额外留白，因此不修改页面内容布局。

## 测试与验证

先新增失败测试，再实现代码：

1. 收起状态活动按钮宽高为 54pt、中心正确；展开和收起状态的活动玻璃外观值完全相同，图标始终为品牌红色。
2. 未选中图标中心在动画全过程保持不变，透明度降为 0，缩放降为 0.92。
3. 普通图标在展开进度达到约 50% 前保持隐藏，之后平滑出现。
4. 滚动行为使用 20pt/16pt 阈值，锁定期间不反向切换且不累积跳变。
5. 现有选中标签、图标顺序和点击展开行为继续通过。
6. 在 iPhone 17 Pro / iOS 26.5 模拟器录制或逐帧检查收起、展开和快速上下滚动。
7. 检查运行日志，不得出现崩溃、布局约束或 Swift 运行时错误。

## 验收标准

- 无粉色横向尾巴、矩形红色块、图标重叠、闪烁或重影。
- 动画中只有一个活动按钮和一个活动背景。
- 收起完成为完整 54pt 浅粉半透明玻璃圆，图标保持品牌红且居中。
- 展开是收起的平滑反向过程，普通图标在胶囊展开后再出现。
- 快速上下滚动不连续触发、不抖动。
- 页面其他视觉、数据、导航和布局保持不变。
