# 页面验收清单

## 全局

- [ ] 原有 Tab 数量、顺序、入口和点击行为保持不变。
- [ ] 底部导航不遮挡滚动内容。
- [ ] 页面支持安全区和不同 iPhone 尺寸。
- [ ] 所有金额、日期、账户、分类数据来自原有模型，不写死为效果图数据。
- [ ] 列表仍可点击、编辑、删除或进入详情，行为与改版前一致。
- [ ] 大字体下没有重要文字截断。
- [ ] 角色插画不截断主要数字，不挡按钮。
- [ ] 当前页面的 Tab 高亮正确。

## 账单

- [ ] 保留收入、支出、结余汇总。
- [ ] 保留搜索与筛选。
- [ ] 保留按日期分组和交易来源。
- [ ] 收入绿色、支出红色，品牌橙色不替代财务语义色。

## 日常账本

- [ ] 日历的日期排列和月份切换逻辑不变。
- [ ] 每日金额显示逻辑不变。
- [ ] 选中日期与下方列表联动不变。
- [ ] 小票、资产、清单、预算入口保留。

## 攒钱

- [ ] 目标金额、已存金额、进度与剩余天数来自真实数据。
- [ ] “存入”操作流程不变。
- [ ] 自动计划与小荷包识别状态逻辑不变。

## 明细

- [ ] 个人/家庭切换逻辑不变。
- [ ] 月份切换与分类筛选逻辑不变。
- [ ] 图表数据与列表合计一致。
- [ ] 图表颜色具备可区分性，不只依赖颜色表达。

## 更多

- [ ] 账户、分类、账本、预算、家庭、备份、导出入口完整。
- [ ] 原有设置项即使效果图未出现，也不能被删除。
- [ ] 头像和家庭账户卡只做视觉升级，不改变隐私逻辑。
